## 使用`react`构建的问答
原型项目为[vue2-happyfri](https://github.com/bailicangdu/vue2-happyfri)，使用`react`重构

使用的技术栈为 `react` + `react-redux` + `react-router`

## 项目运行
```bash
git clone https://github.com/catchonme/react-QA.git

cd react-QA

npm install 

npm start
```
